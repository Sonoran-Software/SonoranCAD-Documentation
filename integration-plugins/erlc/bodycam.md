# Bodycam

## Configuring the Bodycam

### 1. Download the App

Download the Windows or OSX desktop application.

### 2. Select the Roblox Game

Once inside of the police, fire, EMS, or dispatch panel click on your unit header to open the unit editor > **Bodycam Source Window** > **Roblox** > **Start**

Once configured and started, your bodycam will automatically select the same window and start the next time you use the desktop application.
