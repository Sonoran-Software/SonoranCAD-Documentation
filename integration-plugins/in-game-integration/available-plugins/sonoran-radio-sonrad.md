# Sonoran Radio (sonrad)

{% hint style="info" %}
This submodule connects Sonoran CAD to [Sonoran Radio](https://sonoranradio.com/), a separate resource and system.
{% endhint %}

<figure><img src="../../../.gitbook/assets/sonoran_radio_promo (1).png" alt=""><figcaption><p>Sonoran CAD x Sonoran Radio</p></figcaption></figure>

## Activation Guide

### 1. Download and Install the Resource

If you have not already, install the [Sonoran CAD FiveM resource](../fivem-installation/).

Then, [enable the `sonrad` plugin](../submodule-configuration/).

### 2. Adjust the Configuration

The CAD display settings are stored inside of the `/configuration/sonrad_config.lua` file.

### 3. Ensure Players are Linked

Ensure the players have already [linked their CAD](../link-user-in-game.md) for this integration to work.

## Usage

[Learn more about the Sonoran Radio integration here!](https://info.sonoranradio.com/tutorials/integrations/sonoran-cad-integration)
