# Set Penal Codes

<mark style="color:green;">`PUT`</mark> `https://api.sonorancad.com/v2/general/penal-codes`

Replace the community penal code configuration.

## Request Body

```json
{
  "codes": []
}
```

## Response

Returns the number of updated penal code entries.
