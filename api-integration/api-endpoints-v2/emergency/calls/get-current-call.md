# Get Current Call

<mark style="color:green;">`GET`</mark> `https://api.sonorancad.com/v2/emergency/accounts/{accountUuid}/current-call`

Retrieve the currently selected identifier and active call for an account.

## Response

```json
{
  "identId": 15,
  "callId": 1042
}
```
