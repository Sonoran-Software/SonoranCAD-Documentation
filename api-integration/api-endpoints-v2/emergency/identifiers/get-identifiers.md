# Get Identifiers

<mark style="color:green;">`GET`</mark> `https://api.sonorancad.com/v2/emergency/accounts/{accountUuid}/identifiers`

Retrieve the selected identifier and all identifiers for an account.

## Response

```json
{
  "selectedIdent": 15,
  "identifiers": []
}
```
